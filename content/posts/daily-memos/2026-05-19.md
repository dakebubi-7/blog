# 2026-05-19 学习记录

## 今日主要收获

### 1. Claude Code 使用
- Claude Code 是 VS Code 的 AI 编程助手
- 不会保存跨会话的对话历史
- 可以用 `/remember` 命令保存重要信息

### 2. 技术栈
- React 19 + Vite 8
- React Router 处理路由
- Vercel 部署和托管
- GitHub 管理代码

### 3. Git 操作
```bash
git add .
git commit -m "说明"
git push
```
- 遇到冲突可用 `git pull --force` 解决

### 4. Vercel 部署
- `bun run build && bunx vercel --prod` 直接部署到生产环境
- 需要 `vercel.json` 配置 SPA 路由重写
- 可在 Vercel 手动 Redeploy 或 Promote to Production

### 5. 分类功能修复
- 分类页面需要使用 `/category/xxx` 路由
- 修复了 frontmatter 解析问题：`value.trim().replace(/^["']|["']$/g, '')`

### 6. 博客导航栏分类
- 技术 → `/category/技术`
- 随笔 → `/category/随笔`
- AI学习 → `/category/AI学习`

### 7. 遇到的问题
- Git 推送被拒绝 → 用 `git pull --force` 解决
- 分类页显示暂无文章 → 修复 frontmatter 解析
- SPA 路由 404 → 添加 vercel.json
